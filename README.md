# Energyos
Toda a parte do FrontEnd do projeto elétrico Energyos. Atualmente ainda rodando localmente. 
Este é um projeto universitário. Focamos em construir um mínimo produto  viável , caso possível. Este projeto tem apoio de técnnicos da área. O foco do produto é tirar a prova real dos custos, principalmente de áreas residenciais, de toda a parte életrica da nossa região.
